## 🚀 Overview

This project is an interactive **Sorting Algorithm Visualizer** built using React.  
It helps users understand how different sorting algorithms work by visualizing each step of the sorting process in real time.

---

## ✨ Features

- 📊 Visual representation of sorting algorithms  
- ⚡ Real-time animation of sorting process  
- 🎮 Adjustable speed controls  
- 🔄 Random array generation  
- 🎨 Clean and interactive UI  

---

## 🧠 Algorithms Included

- Bubble Sort  
- Selection Sort  
- Insertion Sort  
- Merge Sort  
- Quick Sort  

---

## 🛠️ Tech Stack

- React.js  
- JavaScript (ES6+)  
- HTML5  
- CSS3  
- Vite

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/Bhavanish-Mantri/Sorting-visualizer.git
```
2. Navigate to the project folder:
```
cd Sorting-visualizer
```
3. Install dependencies:
```
npm install
```
4. Run the project:
```
npm run dev
```
---

## 📌 How It Works

The application visualizes sorting algorithms by representing array elements as vertical bars.  
During execution, the algorithms update the array step-by-step, and these changes are reflected in real time to demonstrate comparisons and swaps clearly.
